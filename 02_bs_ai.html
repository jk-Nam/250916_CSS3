창의적인 아이디어를 실용적인 웹 솔루션으로 구현하는 것을 즐기는 백엔드 개발자입니다. 견고하고 확장 가능한 시스템을 구축하는 데 열정을 가지고 있습니다.

프로젝트 보기 GitHub

소개

3년차 백엔드 개발자로서 Python, Java, Node.js를 주로 사용합니다. 클라우드 환경(AWS)에서의 개발 및 배포 경험이 풍부하며, 대용량 트래픽 처리와 데이터베이스 최적화에 관심이 많습니다.

쇼핑몰 백엔드 시스템

사용자 인증, 상품 관리, 주문 처리, 결제 시스템(PG 연동)을 구현했습니다. RESTful API를 설계하고 Django REST framework를 사용했습니다.

주요 기술
Python, Django, DRF, PostgreSQL
Spring Boot
Vue.js
실시간 예약 시스템

동시성 제어를 통해 여러 사용자가 동시에 예약하는 상황을 처리하고, WebSocket을 이용해 실시간 좌석 현황을 제공했습니다.

주요 기술
Java, Spring Boot, JPA, Redis, WebSocket
Node.js
AWS Lambda
서버리스 이미지 처리

AWS S3에 이미지가 업로드되면 Lambda 함수가 트리거되어 이미지 리사이징, 워터마크 추가 등의 작업을 비동기적으로 처리하는 시스템을 구축했습니다.

주요 기술
Node.js, AWS S3, Lambda, API Gateway
